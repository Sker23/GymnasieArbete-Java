# About
The game was created to pass the course gymnasiearbete. It's main feature is the ability to generate a new map for the player every time the game is run. The entire project is written in java. 

# How to use
1. Clone the repository:
git clone https://github.com/YourUsername/YourRepoName.git (github.com in Bing)
2. Open the project in your IDE.
3. Make sure the `src/` and `resources/` folder is marked as a source folder.
4. Run the `Game.java` file (the one containing `public static void main(String[] args)` inside of the `main` package
